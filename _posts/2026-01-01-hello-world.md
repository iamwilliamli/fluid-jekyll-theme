欢迎来到你的新博客！这是一篇示例文章，展示了博客支持的主要功能。

## Markdown 基础

这个博客使用 Markdown 来编写内容。你可以使用各种格式：

- **粗体文本**
- *斜体文本*
- `行内代码`
- [链接](https://example.com)

## 代码高亮

博客支持代码语法高亮：

```python
def hello_world():
    print("Hello, World!")

hello_world()
```

```javascript
const greeting = () => {
  console.log("Hello from JavaScript!");
};
```

## 数学公式

博客支持 MathJax 数学公式：

行内公式：$E = mc^2$

块级公式：

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
